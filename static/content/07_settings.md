Settings Overview

The Settings page in OpenWebUI provides the control center for personalizing and managing your environment. Every configurable option—from UI preferences to model behavior—lives here. Designed with clarity in mind, the settings layout separates controls into well-defined categories such as General, UI Customization, Models, Network, and Extensions.

Within General Settings, you can manage language preferences, default model selection, startup behavior, and theme settings. Model Settings allow you to adjust model parameters such as temperature, max tokens, system prompts, and performance optimizations.

For power users, advanced sections like Network or Developer Tools offer deeper configuration, including API endpoints, local server settings, storage paths, or security preferences.

The Settings panel ensures that every user—beginner or advanced—can shape OpenWebUI into the ideal environment for their workflow. The modular structure makes adjustments intuitive, while safeguards prevent accidental misconfigurations.
