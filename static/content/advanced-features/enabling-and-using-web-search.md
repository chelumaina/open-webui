# Enabling and Using Web Search

LexLuma’s web search integration must be **enabled in settings** to allow AI to access external legal resources for research and referencing.

---

## 🔧 Steps to Enable Web Search

1. Navigate to **Settings > Advanced Features**.  
2. Locate **Web Search Integration** and toggle it on.  
3. Authorize any required access permissions.  
4. Optionally configure **preferred legal sources** or jurisdictions.  

---

## 🖥️ Using Web Search in Chat

- Start a new chat and type your query.  
- LexLuma automatically searches trusted legal websites.  
- Results are summarized with **source citations**.  
- Follow-up questions maintain context for multi-step research.  

---

## 💡 Best Practices

- Specify jurisdiction or document type in queries for accurate results.  
- Combine results with internal document RAG for more comprehensive analysis.  
- Regularly review search preferences to include the latest legal databases.  
- Use **chat history and pinning** to track relevant external references.
