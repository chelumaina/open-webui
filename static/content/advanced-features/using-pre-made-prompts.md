# Using Pre-made Prompts

LexLuma’s **pre-made prompts** provide ready-to-use instructions for common legal tasks. These prompts help streamline research and drafting while ensuring consistent output quality.

---

## 📝 Key Features

- **Legal Document Summaries:** Condense statutes, case rulings, or contracts.  
- **Argument Generation:** Produce persuasive arguments based on legal precedents.  
- **Citation Extraction:** Automatically identify key references from uploaded documents.  
- **Template-Based Drafting:** Create standard legal forms, letters, or memos.

---

## 🔧 How to Use Pre-Made Prompts

1. Navigate to the **Prompt Library** in the sidebar.  
2. Browse categories such as **Statutes, Case Law, Contracts**, or **Opinions**.  
3. Select a prompt suitable for your task.  
4. Apply the prompt to a chat or document query.  
5. Review AI-generated output and make adjustments as necessary.

---

## 💡 Best Practices

- Match prompts to the type of legal task to maximize relevance.  
- Use follow-up queries to refine answers generated from pre-made prompts.  
- Combine pre-made prompts with **RAG-enabled document searches** for precise referencing.  
- Save frequently used prompts to your personal workspace for faster access.
