# Attaching Files to Chats

LexLuma allows you to upload documents directly into a chat so the AI can analyze, summarize, compare, or extract information from them. This feature is essential for legal workflows where source materials like rulings, pleadings, affidavits, statutory extracts, or evidence documents must be reviewed with precision.

---

## 📎 How to Attach Files

You can attach files using either of the following methods:

### **1. Drag and Drop**
Simply drag your document into the chat window. The upload area becomes highlighted, confirming the drop zone.

### **2. Attachment Button**
Click the **📎 Attach** button located beside the message composer. This opens your file browser and lets you select documents manually.

Once uploaded, each file appears as a preview tile with:
- File name  
- Size  
- Optional remove button  
- Status indicator (uploading, ready, or error)

---

## 🔍 Why Attach Documents?

LexLuma uses the attached file to deliver:
- Document summaries  
- Case issue extraction  
- Statutory interpretation  
- Clause-by-clause review  
- Comparison with other uploaded documents  
- Drafting based on document content  
- Extraction of facts, parties, or timelines  
- Detection of inconsistencies or conflicts

Because LexLuma is trained on real legislative and judicial materials, file-based queries are extremely accurate and helpful for litigation and advisory work.

---

## 🛠️ Best Practices When Uploading

- Upload **searchable PDFs** when possible.  
- Break large bundles into parts for faster processing.  
- Use the file name to indicate context (e.g., *Plaintiff Affidavit - 12 May.pdf*).  
- Combine your file with a clear question, e.g.:

Please extract the key legal issues in the attached ruling.


- Upload multiple files if comparison is required.
