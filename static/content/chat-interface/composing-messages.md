# Composing Messages

The message composer in LexLuma is engineered to support detailed, professional legal queries. Whether drafting a statutory interpretation request or uploading exhibits for analysis, the composer provides a flexible space that adapts to your workflow.

---

## ✍️ Drafting Effective Queries

To get the most reliable output:
- Clearly specify the **jurisdiction**, **statute**, or **case name**.  
- Provide **context**: matter type, issue, or questions.  
- Use **structured prompts** for precision.  
- Attach supporting documents before or during drafting.

Example:  
Analyze the attached ruling and identify any inconsistencies with the Evidence Act, Section 63(1)(c).


## ⌨️ Composer Features

### **Multi-line Input**
Press `Shift + Enter` to add new lines—ideal for drafting long legal instructions.

### **Markdown Support**
LexLuma supports headings, bullet points, code blocks, and tables.

### **File Attachment**
Click the 📎 icon or drag files directly into the composer.  
See page: [/chat-interface/attaching-files-to-chats](#)

### **Quick Shortcuts**
| Action | Shortcut |
|--------|----------|
| Send message | Enter |
| New line | Shift + Enter |
| Open attachments | Ctrl/Cmd + U |
| Open Message Actions | Right-click / tap & hold |

---

## 🎯 Tips for Strong Prompts
- Ask one question at a time for maximum precision.  
- Use “Compare”, “Interpret”, “Summarize”, or “Extract Issues” for structured tasks.  
- Include sample text where needed.  
- Upload documents instead of pasting long text.
