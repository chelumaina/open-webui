# Regenerating Responses

The **Regenerate Response** feature allows you to ask LexLuma to produce an alternative answer to a previous query without rewriting or resending the prompt. This is especially useful in legal research workflows, where different interpretations, drafting styles, or analytical depths may be required for the same issue.

---

## 🔄 When Should You Regenerate a Response?

LexLuma’s regeneration capability is valuable for:

### **1. Alternative Legal Reasoning**
Generate multiple perspectives on:
- Statutory interpretation  
- Case law analysis  
- Issue spotting  
- Arguments for both sides  

### **2. Drafting Variants**
Useful when creating:
- Legal opinions  
- Submissions  
- Letters  
- Pleadings in different tones or complexity levels  

### **3. Refining or Elaborating**
If the first response lacks depth, misses context, or needs more detail, regeneration prompts the model to reconsider the prompt and produce improved output.

### **4. Simplified or More Advanced Versions**
Ask for:
- A more concise explanation  
- A more advanced academic analysis  
- A practitioner-focused summary  

This helps accommodate different readers—advocates, clerks, and legislative drafters.

---

## 🛠️ How to Use It

1. Hover over the AI’s message.  
2. Click the **“Regenerate”** icon or select it from the Message Actions Menu.  
3. A fresh answer will be generated directly beneath the previous one.  

You can regenerate multiple times until you get the ideal output.

---

## 📘 Tips for Effective Regeneration

- Combine regeneration with **edited prompts** for more accurate results.  
- Use it to compare multiple viewpoints quickly.  
- If the response still needs improvement, try adding clarifying instructions before regenerating.  

Example:

Regenerate the response but focus only on jurisdictional issues.
