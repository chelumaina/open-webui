# Supported File Types

LexLuma is designed to support the most common document formats used by legal practitioners. Whether you’re submitting rulings, contracts, pleadings, or scanned exhibits, LexLuma can process and analyze a broad range of file types efficiently.

---

## 📂 Primary Supported File Types

### **1. PDF (.pdf)**
- Ideal for court rulings, statutes, contracts, affidavits  
- Supports OCR for scanned documents (quality dependent)

### **2. Word Documents (.doc, .docx)**
- Best for editable legal drafts, submissions, and structured content  
- Preserves formatting during analysis

### **3. Text Files (.txt, .rtf, .md)**
- Great for raw text, notes, and extracted legal sections

### **4. Spreadsheets (.csv, .xlsx)**  
Useful for:
- Financial records  
- Evidence tables  
- Comparative statutory matrices  

### **5. Images (.png, .jpg, .jpeg)**
- Used for scanned pages, handwritten notes, or photos of exhibits  
- LexLuma attempts automatic text extraction using OCR

---

## 🧾 About OCR Documents

While LexLuma supports **OCR (Optical Character Recognition)**, the accuracy depends on:
- Image clarity  
- Arrangement of text  
- Presence of stamps or handwritten markings  

Clean, high-resolution scans generally yield much better results.

---

## ⚠️ Unsupported or Partially Supported Formats

- Audio (mp3, wav)  
- Video (mp4, mov)  
- Executable files (exe)  
- Very large PDFs exceeding system limits  

Unsupported files will show an error after upload.

---

## 📘 Tips for Successful Uploads

- Convert images to PDF for better multi-page processing.  
- Compress overly large PDFs.  
- Use Word format for heavily formatted drafts.  
- Avoid password-protected files (remove protection before upload).
