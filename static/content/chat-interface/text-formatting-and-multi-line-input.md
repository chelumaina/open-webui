# Text Formatting and Multi-line Input

LexLuma supports a rich text formatting environment to help legal professionals draft structured requests that the model can interpret clearly. Whether preparing multi-part queries, outlining issues, or drafting legal templates, proper formatting ensures more accurate AI responses.

---

## 📝 Markdown Support

LexLuma recognizes standard Markdown syntax. This allows you to format complex prompts such as legal arguments, case summaries, or statutory breakdowns.

Examples:

### **Headings**
Issues for Determination
markdown
Copy code

### **Bullet Lists**
Elements of the offence

Procedural irregularities

Burden of proof considerations

markdown
Copy code

### **Numbered Lists**
Background

Analysis

Holding

markdown
Copy code

### **Blockquotes**
Quote applicable sections of statutes or judgments

 

## ⌨️ Multi-line Input for Long Prompts

Press **Shift + Enter** to create a new line without sending the message. This is ideal for:

- Multi-step instructions  
- Draft pleadings  
- Structured legal analyses  
- Detailed comparison tasks  

Example:
Please analyze the following:

Jurisdictional issues

Procedural fairness

Compliance with relevant statutes

 

## 🎯 Tips for Better Formatting

- Break long queries into small, readable sections.  
- Use headings to separate issues.  
- List key facts clearly before asking legal questions.  
- Prefer Markdown tables for structured comparisons.
