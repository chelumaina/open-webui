# Introduction

### Welcome to the OpenWebUI User Guide

OpenWebUI is an intuitive, modern interface designed to help you interact seamlessly with locally hosted or remote AI models. Whether you're a beginner exploring AI chat interfaces or an advanced user managing multiple models and workflows, this guide will help you navigate every corner of the platform with confidence.

This documentation focuses primarily on UI navigation, ensuring you can quickly understand how to move around the interface, customize your workspace, manage models, and access essential tools. Each section is organized to help you learn step-by-step, from understanding the layout to optimizing your full AI experience.

OpenWebUI aims to offer a familiar yet powerful environment—combining elegant UI design, responsive components, and practical features such as chat histories, system prompts, model switching, extensions, and advanced settings.

In this guide, you will:

Get an overview of the main dashboard and layout

Learn how the sidebar, menus, and workspace areas function

Discover how to interact with models through chat or plugins

Understand how to customize your environment for productivity

Explore best practices and troubleshooting tips

If you're setting up OpenWebUI for your team or using it individually, this guide ensures you get the most value from the experience. Let’s begin by understanding how the interface is structured and how each component contributes to smooth and efficient navigation.##
