# Appearance Customization

LexLuma’s appearance settings let you tailor visual aspects of the interface to improve readability and focus during long research sessions. This page explains themes, font size, layout density, and color preferences.

## Customizable Elements

- **Theme Selection:** Light, Dark, or Auto (system-based).  
- **Font Size:** Small, Medium, Large (affects body and code text).  
- **Layout Density:** Compact, Comfortable, Spacious.  
- **Color Scheme:** Accent colors for highlights and tags.  
- **High-Contrast Mode:** For accessibility compliance.

## Accessing Appearance Settings

Navigate to `Settings → Appearance`. Changes apply immediately and can be saved as user defaults.

## Tips for Legal Use

- Use **Dark Mode** during long drafting sessions to reduce eye strain.  
- Select **Large font** for courtroom or presentation review.  
- Use **Compact density** when monitoring many chats or documents on one screen.  
- High-contrast mode aids users working in bright courtroom environments.

## Export & Presentation Considerations

Appearance settings affect screen rendering but not exported document styles (PDF/DOCX exports follow firm templates). For client-facing exports, verify export preview to match firm branding.
