# Color Scheme Preferences

Color scheme controls accent colors, tag palettes, and highlight hues across the LexLuma interface. Thoughtful color choices can improve focus when navigating multiple matters or visualizing document tags.

## Options & Elements

- **Accent color:** Affects buttons, links, and active tab highlights.  
- **Tag colors:** Customize colors for document tags (e.g., statutes, pleadings, evidence).  
- **Highlight color:** For annotated excerpts and search hits.  
- **Contrast presets:** Standard / Soft / High-Contrast.

## How to Update

`Settings → Appearance → Color Scheme` — choose a palette or define custom hex values for accent/tag colors. Save as a default, or apply per-workspace for team settings.

## Best Practices for Legal Teams

- Use consistent tag colors across teams (e.g., red = urgent, blue = statute) to avoid confusion.  
- Use high-contrast palettes for courtroom or accessibility needs.  
- Ensure sufficient contrast between text and background to comply with accessibility standards.
