# Font Size and Layout Adjustments

LexLuma lets you adjust typography and spacing to improve readability during research, drafting, and case review. This page explains options and when to use them.

## Available Settings

- **Font size:** Small / Medium / Large — affects chat text, menus, and document previews.  
- **Line height:** Normal / Comfortable / Wide — helps with long-form reading.  
- **Monospace toggles:** For code blocks or file metadata display.  
- **Layout density:** Compact / Comfortable / Spacious.

## How to Change

1. `Settings → Appearance → Font & Layout`  
2. Adjust sliders or dropdowns, preview changes, and save. Changes are immediate.

## Legal Use Cases

- **Large font** for in-court readings or collaborative reviews.  
- **Compact density** for monitoring many chat threads simultaneously.  
- **Monospace** for clause-by-clause legal comparison or contract diffing.

## Accessibility & Export

Changes help on-screen readability only. Exports retain document template fonts for legal filings. Use large font for on-screen presentations but confirm exported font in preview.
