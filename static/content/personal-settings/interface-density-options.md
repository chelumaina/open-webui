# Interface Density Options

Interface density determines how much information appears on-screen at once. Pick the density that best suits monitoring multiple cases, drafting, or presenting in court.

## Density Modes

- **Compact:** Tight spacing, smaller fonts, more items visible — ideal for monitoring many chats and documents.  
- **Comfortable:** Balanced spacing for typical research and drafting.  
- **Spacious:** Wider margins and larger gaps — ideal for presentations or distraction-free drafting.

## How to Change

`Settings → Appearance → Density` — select Compact, Comfortable, or Spacious. Changes are applied immediately.

## When to Use Each Mode

- **Compact:** When you need to scan case lists or compare many documents.  
- **Comfortable:** Daily research and drafting.  
- **Spacious:** Reviewing long judgments or running presentations with teammates.
