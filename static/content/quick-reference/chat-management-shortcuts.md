# Chat Management Shortcuts

Managing chats efficiently is crucial for legal professionals who need to maintain clarity across multiple matters, client files, and research threads. LexLuma provides a comprehensive set of **chat management shortcuts** that streamline the process of creating, organizing, editing, and navigating active conversations.

These shortcuts help you stay focused on legal analysis rather than tool operations, especially during high-volume research sessions.

---

## ✨ Core Chat Controls

| Action | Shortcut |
|--------|----------|
| Start New Chat | `Ctrl + N` / `Cmd + N` |
| Rename Chat | `Ctrl + R` |
| Delete Chat | `Ctrl + Shift + Del` |
| Duplicate Chat | `Ctrl + Shift + D` |
| Export Chat | `Ctrl + Shift + E` |
| Clear Conversation | `Ctrl + Shift + C` |

These functions allow you to handle research threads with speed—particularly helpful when preparing for hearings or drafting submissions.

---

## 📑 Conversation Navigation & Editing

| Action | Shortcut |
|--------|----------|
| Scroll Up | `Page Up` |
| Scroll Down | `Page Down` |
| Jump to Latest Message | `End` |
| Jump to Top | `Home` |
| Edit Last Message | `↑` (Arrow Up) immediately after sending |
| Delete Last Message | `Ctrl + Backspace` |

Editing the last message is ideal when revising prompts or correcting citations.

---

## 📌 Organization Shortcuts

| Action | Shortcut |
|--------|----------|
| Pin Chat | `Ctrl + Shift + P` |
| Move Chat to Folder | `Ctrl + Shift + M` |
| Archive Chat | `Ctrl + A` |
| Restore Archived Chat | `Ctrl + Shift + A` |

For legal workflows involving multiple matters or clients, these organization shortcuts are essential.
