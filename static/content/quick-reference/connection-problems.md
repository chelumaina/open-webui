# Connection Problems

Connectivity issues can interrupt research sessions. This guide explains how to diagnose and resolve them.

---

## 🌐 Causes of Connection Issues

- Poor internet connectivity  
- VPN or firewall restrictions  
- Browser blocking WebSockets  
- Server undergoing maintenance  

---

## 🛠 Solutions

1. Reload LexLuma  
2. Turn off VPN temporarily  
3. Try another browser (Chrome/Firefox recommended)  
4. Check firewall settings  
5. Verify that WebSockets are enabled  

---

## 🧩 Advanced Tips

- Clear local storage  
- Reset DNS (`ipconfig /flushdns`)  
- Try mobile hotspot to test your connection
