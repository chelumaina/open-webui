# Connection Status Indicators

Connection status indicators help you understand the real-time availability of LexLuma’s services, including backend connectivity, model responsiveness, and sync status. Reliable connectivity is crucial, especially for legal practitioners who depend on up-to-date data retrieval and real-time AI reasoning.

---

## 🌐 Connection Icons

| Icon | Status |
|------|--------|
| 🟢 **Online** | Full functionality available |
| 🟡 **Limited Connection** | Some features may be delayed |
| 🔴 **Offline** | No interaction possible |
| ⚪ **Reconnecting…** | System attempting to restore service |

---

## 🔄 Sync Status

Some operations require synchronization, such as:

- Chat history syncing  
- Workspace saving  
- Document indexing  
- Prompt preset updates  

| Icon | Meaning |
|------|---------|
| 🔁 | Sync in progress |
| ✔️ | Sync complete |
| ⚠️ | Sync failed |

---

## 🛠 Troubleshooting Tips

- Check your internet connection  
- Reload LexLuma (`Ctrl + R`)  
- Log out and back in  
- Check firewall or VPN restrictions
