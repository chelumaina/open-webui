# Feature-Specific Troubleshooting

Some issues are tied to specific features such as RAG, web search, multimodal analysis, or voice tools.

---

## 🧠 RAG/Document Chat Issues

**Problem:** AI cannot read uploaded PDF  
**Solution:**  
- Ensure PDF has a text layer  
- Use OCR before upload  
- Re-upload after refreshing page  

---

## 🌐 Web Search Issues

**Problem:** No results returned  
**Fix:**  
- Check API key  
- Verify search is enabled  
- Try different query terms  

---

## 🖼️ Image Analysis Issues

- Ensure file is PNG/JPG  
- Avoid blurry scans  
- Crop unnecessary sections  

---

## 🎤 Voice Tools

- Check microphone permissions  
- Reduce background noise  
- Use supported audio formats
