# File Upload Issues

Uploading legal files is essential for RAG-based document analysis. Here’s how to resolve upload failures.

---

## ❌ Common Upload Errors

- File too large  
- Unsupported format  
- Scanned PDF without text layer  
- Browser security settings blocking uploads  
- Temporary server processing delays  

---

## 🛠 Fixes

- Compress oversized PDFs  
- Convert images to PDF or PNG  
- Run OCR (if text missing)  
- Disable conflicting browser extensions  
- Upload files one at a time  
- Check your organization’s upload limit
