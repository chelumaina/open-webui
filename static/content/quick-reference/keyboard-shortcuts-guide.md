# Keyboard Shortcuts Guide

The **Keyboard Shortcuts Guide** provides a fast, comprehensive overview of the key combinations that help you work more efficiently in LexLuma. Whether you're navigating chats, editing text, managing prompts, or switching models, this quick reference sheet keeps everything at your fingertips. Mastering shortcuts significantly speeds up workflows, especially during research sprints, hearings, and rapid drafting sessions.

---

## 📌 Why Use Keyboard Shortcuts?

Keyboard shortcuts offer:

- **Increased speed** — complete actions in seconds.  
- **Reduced mouse usage** — ideal for long drafting sessions.  
- **Accessibility benefits** — supports users who rely on keyboard navigation.  
- **Consistent workflow** — the same across devices and operating systems.

➡️ [Navigation Shortcuts](./navigation-shortcuts.md)

## 🔧 Universal Shortcuts

| Action | Shortcut |
|-------|----------|
| Open Command Palette | `Ctrl + K` / `Cmd + K` |
| Search | `Ctrl + F` / `Cmd + F` |
| New Chat | `Ctrl + N` / `Cmd + N` |
| Close Panel | `Esc` |
| Toggle Sidebar | `Ctrl + B` / `Cmd + B` |

These shortcuts work across all interface modes and layouts.
