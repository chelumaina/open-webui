# Message Action Icons

LexLuma provides message-level control tools to enhance usability and support efficient editing, sharing, and regeneration. These icons allow precision control when interacting with long legal responses.

---

## 🧰 Message Action Toolbar

| Icon | Function |
|------|----------|
| ✏️ Edit | Modify your previous message |
| 🔁 Regenerate | Request a new answer |
| 📄 Copy | Copy message content |
| 📎 Attach | Add supplementary files |
| 📤 Export | Save message as PDF/Markdown |
| 🗑️ Delete | Remove message |
| ⭐ Save | Add message to prompt library |

---

## 🧩 Advanced Tools

Certain messages support additional features:

| Icon | Meaning |
|------|---------|
| 🧠 Cite | Generate citations |
| 🧩 Reference Mode | Analyze message cross-links |
| 🗂️ Add to Workspace | Save message to folder |
