# Model Status Indicators

Model status indicators help you understand whether the current AI model is ready, processing, overloaded, or encountering errors. This is essential when working with large legal documents, case summaries, or complex constitutional interpretation tasks.

---

## 🧠 Model Indicator States

| Indicator | Meaning |
|-----------|---------|
| 🟢 **Ready** | Model is available and responsive |
| 🟡 **Processing** | Model is generating or reviewing input |
| 🔵 **Loading** | Model is being initialized |
| 🔴 **Unavailable** | Model offline or error |
| 🟣 **Queued** | Awaiting processing |

---

## ⚙️ Model Load Conditions

You may see:

- “High Load” – Model is busy; responses may be slower  
- “Optimizing” – Model analyzing context window  
- “Warm Start” – Preparing legal RAG index  

These ensure accuracy when interacting with large batches of legal content.
