# Navigation Shortcuts

Efficient navigation is essential for legal professionals who move rapidly between chats, documents, research threads, and model configurations. LexLuma provides a comprehensive set of **navigation shortcuts** to streamline your workflow and reduce reliance on the mouse. Whether you’re managing multiple research sessions or switching between legislative sources, these shortcuts help you maintain flow and reduce cognitive load.

This guide pairs well with:  
➡️ [Keyboard Shortcuts Guide](./keyboard-shortcuts-guide.md)  
➡️ [Chat Management Shortcuts](./chat-management-shortcuts.md)

## 🧭 Core Navigation Shortcuts

These shortcuts help you move quickly across the main sections of the LexLuma interface:

| Action | Shortcut |
|--------|----------|
| Open Sidebar | `Ctrl + B` / `Cmd + B` |
| Focus Search Bar | `Ctrl + F` / `Cmd + F` |
| Jump to New Chat | `Ctrl + N` / `Cmd + N` |
| Go to Home Dashboard | `Ctrl + H` / `Cmd + H` |
| Navigate to Settings | `Ctrl + ,` / `Cmd + ,` |

---

## 📂 Moving Between Chats

For users juggling multiple ongoing matters, these shortcuts are essential:

| Action | Shortcut |
|--------|----------|
| Next Chat | `Ctrl + ]` |
| Previous Chat | `Ctrl + [` |
| Duplicate Chat | `Ctrl + Shift + D` |
| Open Chat Search | `Ctrl + Shift + F` |

These let you shift rapidly between research threads without breaking concentration.

---

## 🧱 Workspace Navigation

Workspaces help legal teams categorize research—for example, “Criminal Procedure,” “Contracts,” or “Land Disputes.” Use these shortcuts to navigate them:

| Action | Shortcut |
|--------|----------|
| Open Workspace List | `Ctrl + Shift + W` |
| Switch to Next Workspace | `Ctrl + Alt + →` |
| Switch to Previous Workspace | `Ctrl + Alt + ←` |

Learn more here:  
➡️ [Workspaces Overview](../sidebar-navigation/workspaces-overview.md)

## 🖥️ Panel & View Controls

| Action | Shortcut |
|--------|----------|
| Toggle Split View | `Ctrl + Shift + S` |
| Expand/Collapse Right Panel | `Ctrl + Shift + E` |

| Zoom Out | `Ctrl + -` / `Cmd + -` |

These shortcuts help tailor the interface for long legal drafting sessions.
