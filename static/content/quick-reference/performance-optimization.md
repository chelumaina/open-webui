# Performance Optimization

For demanding legal workflows, maximizing system responsiveness ensures smooth research sessions.

---

## 🚀 Ways to Improve Speed

### 1. Reduce Context Window
Clear or truncate old messages to optimize speed.

### 2. Use Appropriate Model
Lighter models respond faster for simple tasks.

### 3. Close Heavy Browser Tabs
Limit competing processes.

### 4. Upload Smaller Files
Large case files increase processing time.

### 5. Disable Web Search when unnecessary

---

## 🧠 Model Optimization Tips

- Lower temperature for predictable outputs  
- Shorten response length  
- Use specialized models for legal reasoning
