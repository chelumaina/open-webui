# Status and Notification Icons

Notifications help you stay updated on system events, AI actions, document readiness, and account alerts.

---

## 🔔 Notification Types

| Icon | Meaning |
|------|---------|
| 🔔 | Standard notification |
| ⚠️ | Warning (e.g., large file) |
| ❗ | Error (connection or parsing) |
| 🔄 | Background task |
| 📥 | Download ready |
| 📤 | Export completed |

---

## 📡 System Alerts

| Icon | Meaning |
|------|---------|
| 🌐 | Web search active |
| 📶 | Connection restored |
| ⚡ | High load detected |
