# Status Indicators Reference

LexLuma uses a clean system of visual indicators to help you understand system status, model readiness, file uploads, and connection health. This reference guide helps you quickly identify what each color or icon means, reducing confusion and improving workflow efficiency.

Status indicators are particularly important for legal professionals working under pressure—ensuring the system is ready before proceeding.

---

## 🟢 General Indicator Categories

LexLuma uses four main categories of indicators:

1. **Connection Indicators**  
2. **Model Status Indicators**  
3. **Upload & Processing Indicators**  
4. **Notification & Alert Icons**

Each category is color-coded and icon-based for fast recognition.

---

## 🎛️ Color Codes

| Color | Meaning |
|--------|---------|
| **Green** | Fully operational |
| **Yellow** | Waiting / Processing / Limited functionality |
| **Red** | Error or critical issue |
| **Grey** | Idle or inactive |
