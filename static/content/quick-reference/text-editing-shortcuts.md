# Text Editing Shortcuts

LexLuma's text input area supports advanced editing shortcuts to help lawyers, clerks, and researchers craft precise queries. These shortcuts mimic common productivity tools, enabling quick corrections, formatting enhancements, and structured prompt building.

Useful for drafting legal instructions, summarizing long rulings, or guiding the AI through multi-step reasoning.

---

## 📝 Basic Editing Shortcuts

| Action | Shortcut |
|--------|----------|
| Undo | `Ctrl + Z` / `Cmd + Z` |
| Redo | `Ctrl + Shift + Z` / `Cmd + Shift + Z` |
| Cut | `Ctrl + X` / `Cmd + X` |
| Copy | `Ctrl + C` / `Cmd + C` |
| Paste | `Ctrl + V` / `Cmd + V` |

These allow effortless correction and modification during prompt drafting.

---

## 🔤 Text Selection Controls

| Action | Shortcut |
|--------|----------|
| Select All | `Ctrl + A` / `Cmd + A` |
| Select Word | `Ctrl + Shift + →` |
| Select Line | `Shift + Home` or `Shift + End` |
| Select Paragraph | `Ctrl + Shift + ↑ / ↓` |

Fast selection improves prompt efficiency when revising instructions.

---

## 🧱 Multi-Line Formatting

| Action | Shortcut |
|--------|----------|
| New Line | `Shift + Enter` |
| Insert Code Block | ``` Ctrl + Shift + ` ``` |
| Toggle List | `Ctrl + Shift + L` |
| Indent | `Tab` |
| Outdent | `Shift + Tab` |

Multi-line formatting is especially useful for structured legal prompts (IRAC, multi-question prompts, enumerated tasks).
