# Toolbar Icons Reference

The toolbar contains quick-access icons that streamline navigation and improve workflow efficiency. This guide explains each icon's function for lawyers, clerks, and researchers.

---

## 🧭 Core Toolbar Icons

| Icon | Function |
|------|----------|
| ➕ New Chat | Create a fresh conversation |
| 🔍 Search | Search chats, documents, or workspaces |
| ⚙️ Settings | Personal and system configuration |
| 🌐 Web | Enable or disable web search mode |
| 📄 Upload | Add files for document chat |
| 👤 Profile Menu | Account and profile options |

---

## 📚 Workspace Tools

| Icon | Function |
|------|----------|
| 📂 Open Workspace | Show workspace list |
| 🗂️ Manage Workspace | Rename, move, or organize |
| 📌 Pin Workspace | Keep workspace visible |

---

## 🧠 Model Controls

| Icon | Function |
|------|----------|
| 🧠 Models | Open model selector |
| ⚙️ Model Settings | Adjust temperature, length, etc. |
| 🔄 Reload Model | Refresh model session |
