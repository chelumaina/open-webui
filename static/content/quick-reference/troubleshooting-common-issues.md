# Troubleshooting Common Issues

LexLuma is designed for stability, but issues may occur due to network errors, heavy documents, API limits, or browser conflicts. This guide helps resolve the most common problems quickly.

---

## 🛠 Frequent Issues and Solutions

### 1. System Running Slowly
- Close unused tabs  
- Clear browser cache  
- Reduce chat context length  
- Check model performance settings  

See: [Performance Optimization](./performance-optimization.md)

### 2. Web Search Not Working
- Ensure web search API key is configured  
- Verify internet connection  
- Check your plan’s search limits  

See: [Web Search API Configuration](../personal-settings/web-search-api-configuration.md)

### 3. Chats Not Loading
- Force refresh (`Ctrl + R`)  
- Log out and log back in  
- Check service status
