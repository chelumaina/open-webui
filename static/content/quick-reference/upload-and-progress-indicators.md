# Upload & Progress Indicators

Uploading legal documents—PDFs, pleadings, judgments, statutes, or scanned materials—is a core part of LexLuma. This page explains the meaning of each upload and processing indicator to help you track document readiness.

---

## 📤 Upload Status Icons

| Icon | Meaning |
|------|---------|
| ⏳ | Upload in progress |
| 📁 | File queued |
| ✔️ | Upload complete |
| ❌ | Upload failed |
| 🔄 | Retry available |

---

## 🔍 Processing Indicators

After upload, LexLuma may:

- Extract text  
- Parse sections  
- Index content for RAG  
- Run OCR on scanned pages  

Icons:

| Icon | Meaning |
|------|---------|
| 🧠 | AI reading the document |
| 🏷️ | Metadata extraction |
| 📚 | Adding to knowledge base |
| 💬 | Ready for chat |
