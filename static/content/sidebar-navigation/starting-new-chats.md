# Starting New Chats

Creating a new chat in LexLuma is the first step toward initiating a structured, AI-assisted research session. Whether you're analyzing legislation, reviewing a case, or exploring conflicting authorities, each new chat allows you to maintain separation and clarity in your work.

---

## 🚀 How to Start a New Chat

From the **left sidebar**, click:

### **➕ New Chat**

A fresh workspace loads with a clear text input area. Here you can:

- Ask research questions  
- Paste legal excerpts  
- Upload documents for analysis  
- Begin multi-step conversations  

Each chat is automatically saved, timestamped, and added to your Chats Section.

---

## 🧠 What Happens When You Create a New Chat?

LexLuma initiates a clean conversation context using its RAG-enhanced legal knowledge base. This ensures:

- Only relevant information is considered  
- Each matter remains isolated  
- Document uploads remain tied to that thread  

This is especially important when handling multiple briefs or advising several clients simultaneously.

---

## 💼 When Should You Start a New Chat?

Start a new chat when:

- You begin research on a new matter  
- You want to isolate legal issues (e.g., “Jurisdiction,” “Evidence,” “Sentencing”)  
- You upload documents for a case  
- You need a clean environment for drafting legal arguments  

Keeping topics separate mirrors how legal practitioners maintain case files.
