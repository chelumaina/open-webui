# Contact Support

Need help with OpenWebUI? We're here to assist you! Choose the best way to reach us based on your needs.

## Support Channels

### Community Support

**Discord Community**
- Join our active Discord server for real-time help
- Connect with other users and share experiences
- Get quick answers from community members
- [Join Discord Server →](https://discord.gg/openwebui)

**GitHub Discussions**
- Ask questions and share ideas
- Browse existing discussions
- Get help from maintainers and contributors
- [Visit GitHub Discussions →](https://github.com/open-webui/open-webui/discussions)

### Report Issues

**GitHub Issues**
- Report bugs and technical issues
- Request new features
- Track issue status and updates
- [Open an Issue →](https://github.com/open-webui/open-webui/issues)

**Bug Report Guidelines**
- Search existing issues first
- Provide detailed reproduction steps
- Include system information and logs
- Attach screenshots if applicable

### Email Support

For business inquiries, partnerships, or sensitive matters:

**General Inquiries**
- Email: support@openwebui.com
- Response time: 24-48 hours

**Business & Enterprise**
- Email: enterprise@openwebui.com
- For licensing and partnership inquiries

**Security Issues**
- Email: security@openwebui.com
- Please report security vulnerabilities privately

## Before You Contact Us

To help us assist you better, please:

1. **Check the Documentation**
   - Browse our [Help Center](/help)
   - Review [FAQs](/help/faqs)
   - Check [Troubleshooting Guide](/help/troubleshooting)

2. **Prepare Information**
   - OpenWebUI version
   - Browser and OS information
   - Steps to reproduce the issue
   - Error messages or logs
   - Screenshots (if applicable)

3. **Search Existing Resources**
   - [GitHub Issues](https://github.com/open-webui/open-webui/issues)
   - [GitHub Discussions](https://github.com/open-webui/open-webui/discussions)
   - [Community Discord](https://discord.gg/openwebui)
