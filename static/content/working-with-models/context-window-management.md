# Context Window Management

LexLuma allows users to manage the **context window**, controlling how much previous conversation or document content the AI can consider when generating responses. Proper context management ensures accurate multi-step reasoning and coherent legal analysis.

---

## 🔍 What is a Context Window?

- Defines the number of **tokens or characters** the model remembers in a conversation.  
- Larger context windows support:
  - Complex multi-turn queries  
  - Case law comparisons  
  - Document summaries spanning multiple sections  
- Smaller context windows are faster and suitable for short, focused queries.

---

## 🔧 How to Adjust

1. Open the **Model Settings Panel**.  
2. Locate the **Context Window Setting**.  
3. Increase or decrease based on task requirements.  
4. Apply changes immediately to the current chat session.

---

## 💡 Best Practices

- Use larger windows for **long legal documents** or multi-step research tasks.  
- Use smaller windows for **quick queries** to speed up response times.  
- Combine context window adjustments with **response length** and **temperature** for optimal output.  
- Regularly clear irrelevant chat history to maintain efficiency.
