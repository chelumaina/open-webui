# Response Length Settings

LexLuma enables users to define **maximum response length** for AI-generated outputs. This feature ensures responses are either concise for quick reference or detailed for comprehensive legal research.

---

## 📏 How Response Length Works

- Determines the **maximum number of tokens** the model uses per response.  
- Short responses are suitable for:
  - Quick summaries  
  - Issue identification  
  - Case references  
- Long responses are ideal for:
  - Detailed legal analysis  
  - Drafting opinions or memos  
  - Multi-step argumentation

---

## 🔧 Adjusting Response Length

1. Open the **Model Settings Panel**.  
2. Find the **Response Length Slider** or input box.  
3. Choose the desired maximum length.  
4. Apply the changes to your current chat or save as default for future sessions.

---

## 💡 Best Practices

- For court decision summaries, set moderate lengths for clarity.  
- When drafting legal arguments, set longer lengths to avoid truncated outputs.  
- Combine with **temperature controls** for balanced creativity and depth.  
- Monitor outputs; overly long responses may require regeneration for refinement.
